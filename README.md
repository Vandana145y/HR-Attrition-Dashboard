# HR-Attrition-Dashboard
# HR Employee Attrition Dashboard
# Project Overview

This project is an interactive HR Analytics Dashboard built using Streamlit, Pandas, Plotly, and Machine Learning.

It analyzes employee data to understand attrition patterns and also predicts whether an employee is likely to leave the company.

# Objective

Analyze employee attrition trends

Identify key factors affecting employee retention

Visualize HR data using interactive charts

Build a machine learning model to predict attrition

Provide real-time dashboard insights
